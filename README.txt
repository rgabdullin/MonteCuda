Вычисляется интеграл от функции sin(x) * ln(1 + x) на отрезке [0;pi/2].

./CPU_program [<размер_выборки>]
размер_выборки больше нуля(по умолчанию 64*1024*1024).

./GPU_program [<число_нитей> [<размер_выборки>]]
число_нитей должно быть кратно 256(по умолчанию 16*1024). Если не кратно, то будет увеличено до ближайшего кратного. 
размер_выборки должен быть кратен число_нитей. Если не кратно, то будет увеличен до ближайшего кратного.
